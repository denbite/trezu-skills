# Collection of skills for Trezu

An agent skill that turns incoming invoices into USDT payment proposals on a [Trezu](https://trezu.app) treasury (Sputnik DAO) on NEAR mainnet via `near-cli-rs`. Including setup guide that helps to create a proposals-only implicit account, fund it, and register it as a Requestor member of the treasury.

## Installation

```bash
npx skills add denbite/trezu-skills
```
